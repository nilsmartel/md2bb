# md2bb

CLI Tool to convert markdown files into old-school BBCode codes.

If you don't know what bbcodes are, you probably are never going to need them.
They used to be more popular 10-20 years ago.

```
Usage: md2bb [FILE]
       cat [FILE] | md2bb 

Convert Markdown to BBCode.

Arguments:
  FILE       Path to a Markdown file to convert.
             If omitted, input is read from stdin.

Options:
  -h, --help  Print this help message and exit.
```
